## Getting Started

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm, yarn, or pnpm

### Installation

```bash
git clone https://github.com/dougdevitre/STL-Platform-Advocate.git
cd STL-Platform-Advocate
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
